## ⚡ Technical Architecture

This portfolio was engineered from scratch without relying on heavy frontend frameworks like React or Vue, proving a deep understanding of the DOM, modern CSS, and vanilla JavaScript optimization.

* **Structure:** Semantic HTML5
* **Styling:** Tailwind CSS via CDN for utility-first, responsive design.
* **Animations:** * `AOS (Animate On Scroll)` for buttery-smooth viewport entry reveals.
    * `Typed.js` for flawless, overlapping-free typography loops.
    * Custom CSS `@keyframes` for hardware-accelerated 3D hover physics, glowing box-shadows, and background IDE scrolling.
* **Hosting:** GitHub Pages (CI/CD)

## 🚀 Featured Work Displayed

The portfolio highlights several full-stack and IoT projects, including:

1.  **TransitPay Ecosystem:** An offline-resilient Progressive Web App (PWA) and ACID-compliant Spring Boot ledger designed to replace cash handling on rural transit routes. Features a background sync queue and native hardware QR scanning.
2.  **IoT Smart Auth System:** An end-to-end data pipeline from ESP32-CAM facial-recognition sensors to a centralized Java server triggering physical solenoid locks.
3.  **Pipeline Tracker API:** A complete CRUD database application managing recruitment pipelines with dynamic frontend sorting.
4.  **TNSTC App Redesign:** High-fidelity UI/UX component architecture built in Figma.
